DavinciResolve Organizer
Description

Este script Python automatiza a criação de uma estrutura de pastas profissional e padronizada para projetos no DaVinci Resolve. 
Desenvolvido para rodar diretamente no navegador, ele elimina a tarefa repetitiva de organizar diretórios manualmente, garantindo 
que editores e coloristas mantenham um workflow consistente desde o primeiro dia.

. Interface intuitiva que funciona em qualquer navegador moderno.
  

A Python-based utility designed to automate the creation of professional folder hierarchies for DaVinci Resolve projects. Running 
seamlessly in any web browser, this script eliminates manual organization tasks, ensuring a  clean and consistent workflow 
for editors and colorists.

. Browser-Compatible: User-friendly interface that runs in any modern web browser.



Dieses Python-Skript automatisiert die Erstellung einer professionellen und standardisierten Ordnerstruktur für DaVinci Resolve-Projekte. 
Das Tool läuft direkt im Browser und macht die manuelle Organisation von Verzeichnissen überflüssig, sodass Editoren und Coloristen von Beginn 
an einen konsistenten Workflow beibehalten.


. Browser-Unabhängig: Intuitive Benutzeroberfläche, die in jedem modernen Webbrowser funktioniert.
